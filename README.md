Here is the repository for the Responsive Image Sizing assignment in Web Development. Located inside this repsoitory is three images.
The images here are designed to be resizable based on the width of the browser window.
The first image is half of an image of London, UK from the River THames at night. Only the London Eye is visible in the photo when
the window is at smaller sizes, but at larger sizes, the image shows the Houses of Parliament.
The second image scales naturally with CSS and is of the Eiffel Tower in Paris, France.
The final image is New York City at night, and as the window size gets bigger, the image will be swapped out with higher resolution versions.
